Typography

Typography styles help you to use different header sizes, line-heights wihtout tweaking the css much.

We provide 5 different headers, 5 different font-weights and 5 different line-heights.

Usage

Use the class type1 on a h1 tag or p tag to use these components.

Additionally use can use our extra classes fw-light fw-regular fw-semibold fw-bold
fw-black to get font-weights

Also you can use class text-center to center the text.

You can also use class text-gray for gray text.
