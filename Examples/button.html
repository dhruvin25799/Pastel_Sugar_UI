Button

Buttons are extremely necessary for actions and clicks.

We provide you with 5 types of buttons : Primary, Primary-Outline, Link, Icon Button and FAB.

Usage

Use the class btn to a button tag to use this style.

Aditionally you can use pre-defined classes btn-primary btn-primary-outline
btn-link btn-icon and btn-fab to use our pre-defined styles.
