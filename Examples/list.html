List

Lists are used to display content in an ordered manner. Also can be used for notifications.

We provide you with 2 different lists, spaced list and stacked list and notifications.

Usage

Use the class list-spaced on a ul/ol tag to use a spaced list.

Use the class list-stacked on a ul/ol to use a stacked list.

Additionally use can use our extra clasees no-style to disable numbers/bullets for the list.
