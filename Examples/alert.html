Alert

An alert box is used to display important updates to the user like click events.

We provide you with 4 types of alert boxes : Error, Success, Warning and Info.

Error! Wtf are you doing man!

Success! Wow you're great!

Warning! Bruv, I see you!

Info! PS : We see you!

Usage

Use the class alert on a div to use this component.

Additionally use can use our extra classes error success warning info
to get colored alerts.
