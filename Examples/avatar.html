Avatar

An avatar can be used to show a user's profile picture on various places.

We provide you with 4 types of avatar sizes : Standard (3rem), Medium (4rem), Large (6rem), X-Large (8rem).

Usage

Use the class avatar on an img tag to get the avatar component.

Additionally you can use classes medium large xlarge to get different sizes.
